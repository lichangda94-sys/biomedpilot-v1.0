# Meta Analysis E2E Example Project

This directory contains small source fixtures for the Meta Analysis internal beta validation path.

Status: Developer Preview / testing.

The committed files are inputs only. Generated outputs such as analysis results, figures, reports, export ZIPs, and reproducibility packages are created in temporary test projects and are not committed.

Included inputs:

- `inputs/mock_literature.csv`: compact local literature records used by the Stage M E2E validation test.
- `inputs/mock_literature.ris`: companion RIS fixture for external importer compatibility checks.
- `manifest.json`: describes the intended local validation chain and expected testing-only status.
