# Meta Analysis Realistic Project Fixture

This fixture is for Stage W internal beta validation only.

The source records are PubMed-derived metadata for hydroxychloroquine COVID-19 randomized trials. Long abstracts are intentionally omitted from committed files to keep the fixture compact. Extraction 2x2 values used by tests are manually seeded validation data and must not be interpreted as published trial results.

Use this fixture to validate:

- literature import from CSV/RIS
- screening and full-text status flow
- structured extraction and quality testing workflow
- analysis-ready dataset and OR meta-analysis
- forest/funnel plots
- report, manifests, and reproducibility package generation

Status: Developer Preview / testing. Not production evidence synthesis output.
